# Automatic License Plate Recognition Using OpenCV and Tesseract OCR

This project implements an Automatic License Plate Recognition (ALPR) system using OpenCV for image processing and Tesseract OCR for character recognition. The system detects vehicle license plates from images, extracts the plate region through image preprocessing techniques, and converts the plate characters into editable text using Optical Character Recognition (OCR).

Features
Vehicle license plate detection
Image preprocessing using OpenCV
Edge detection and contour analysis
License plate localization and extraction
Character recognition using Tesseract OCR
Real-time and image-based recognition support
Simple and lightweight implementation
Technologies Used
Python
OpenCV
NumPy
Tesseract OCR
Image Processing Techniques
Working Flow
Input vehicle image.
Convert image to grayscale.
Apply filtering and edge detection.
Detect contours to locate the license plate.
Extract the license plate region.
Perform OCR using Tesseract.
Display recognized license plate number.
Applications
Smart Parking Systems
Toll Collection Systems
Traffic Monitoring
Vehicle Access Control
Security and Surveillance Systems
Future Enhancements

Deep learning-based plate detection
Multi-language license plate support

Improved OCR accuracy under challenging conditions
Output

The system automatically identifies the license plate from a vehicle image and displays the recognized registration number with high accuracy.
